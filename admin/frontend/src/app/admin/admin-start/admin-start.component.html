Rentrée

Procédure de rentrée 📚

Cette petite marche à suivre, va vous guider dans la mise en oeuvre de chaque rentrée. Vous pouvez voir à n'importe quel moment où vous en êtes en cliquant sur chaque étapes.

Cette procédure prendra environ 30 minutes.

Configuration du matériel 📺

Dans cette étape, nous allons ajouter toutes les pointeuses et les configurer

1.1) Installation des pointeuses 📇

Pour chaque machine, ouvrez le boitier et connectez-y un clavier et une souris.

Suivez la procédure Installation de la pointeuse disponible sous l'onglet mise en route du classeur.

1.2) Création des paramètres 💬

Créez les vacances + jours fériés ainsi que les horaires pour l'année courante.

Connectez-vous sur chaque pointeuse en cliquant dessus et choissisez vos paramètres et préférences.

Importation des stagiaires 🚶

Dans cette étape, nous allons importer les stagiaires. Afin de réaliser cette étape correctement, il convient d'utiliser le modèle import.csv disponible avec la documentation du projet.

Répartition par machine 🔠

Sélectionnez où vous voulez placer les stagiaires.

🌈 Vous êtes paré pour la rentrée 🎩
